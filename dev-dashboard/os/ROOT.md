## R-1. S++とは何か（永久固定）

**正式名称：** S++（変更禁止）

**定義：**
「実戦トレーダーKが案内する、信頼ベースのFX・プロップ情報プラットフォーム」

**禁止思想：**
```
❌ AI量産比較サイト
❌ ランキング屋
❌ 煽りアフィリエイトサイト
❌ 「ズレを完全に消すこと」を目的にする（正：検知・補正）
```

**本質：**
```
✅ Kの一次情報・実測データを核にした検証型金融メディア
✅ support-logという実録DBを持つ観測型プラットフォーム
✅ 将来的にツール・AI機能・会員機能を統合する開発プラットフォーム
```

## R-2. 中核循環

```
実測データ取得 → support-log蓄積 → 記事反映
→ ユーザー信頼獲得 → さらなる実測データ取得（循環）
```

この循環を止めないことが最優先。
