# サイトID・タグライン・フェーズ管理ポリシー

最終更新：2026-05-10

---

## 正式サイト名（永久固定・変更禁止）

```
S++
```

**Claudeへの注意：** サイト名「S++」は絶対に変更しない。
タイトルタグ・H1・ナビロゴ全てで「S++」を使用する。

---

## 変更禁止項目

以下は本番・開発環境を問わず変更禁止：

| 項目 | 固定値 |
|---|---|
| 正式サイト名 | S++ |
| brokers URL構造 | /brokers/ |
| prop-firms URL構造 | /prop-firms/ |
| support-log URL構造 | /support-log/ |
| compare URL構造 | /compare/ |
| guides URL構造 | /guides/ |

---

## 変更可能項目（フェーズで変化）

以下はフェーズに応じて変更可能：

- トップページ title
- H1テキスト
- meta description
- タグライン
- LPコピー・ヒーローセクションのコピー
- ナビゲーションメニューの内容

## フェーズ別タイトル方針

### Phase 1（現在）
```
S++｜海外FX・プロップファーム実測ガイド
```
**移行条件：** 初期公開状態（現在このフェーズ）

### Phase 2
```
S++｜FX業者・プロップファーム実測ガイド
```
**移行条件：** 国内FX証券会社記事が最低限公開されたら

### Phase 3
```
S++｜FXトレーダーのための統合プラットフォーム
```
**移行条件：** 以下が全て揃ったら
- ロット計算ツールが実用レベル
- 損益期待値シミュレーターが実用レベル
- トレードジャーナルが最低限動作
- support-logが継続運用中
- 国内FX / 海外FX / プロップファーム記事が揃っている

---

## Phase 3移行前の注意

「FXトレーダーのための統合プラットフォーム」というタイトルを使用する条件を満たす前に変更しないこと。

**Claudeへの注意：**
本番反映時にtitleやH1を確認する際、現在のフェーズ（Phase 1）のタイトルと一致しているかをチェックすること。Phase 3のタイトルを使用している場合は警告を出すこと。

---

## 本番反映前チェック（サイトID関連）

```
□ サイト名がS++で固定されている
□ タグラインが現在フェーズと一致している
□ title / H1 / meta description が一致している
□ 未完成機能を前面に出していない
□ 「統合プラットフォーム」と名乗る条件を満たしている（Phase 3のみ）
□ URL構造を変更していない
```
