## 金融系の表現リスク管理

以下の表現は使用禁止（AI禁止事項より）：

| 禁止表現 | 理由 |
|---|---|
| 「完全に安全」 | 金融系で保証はできない |
| 「必ず儲かる」 | 投資助言に該当する可能性 |
| 「出金拒否はない」（断定） | S++が全事例を網羅できない |
| 「初心者でも安心」 | 根拠なしの断定 |
| 「おすすめです」（根拠なし） | AI臭 |

## DEV ONLYコメントの管理方針

```html
<!-- [DEV ONLY] アフィリエイトリンクに差し替え予定 -->
```

- 本番ではHTMLコメントとして残す（表示されない）
- または本番ビルド時にスクリプトで除去する
- 絶対にコメントアウトを外したまま公開しない
